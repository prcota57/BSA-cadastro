Backup completo

Baixa um arquivo com todos os alunos, mensalidades, presenças, avaliações físicas, reposições e leads de captação de uma vez só.

O nome do arquivo é sempre o mesmo — ao salvar no mesmo lugar de novo, o iPhone pergunta se quer substituir o anterior, então não fica acumulando arquivo à toa.

Esse backup só tem os dados (alunos, financeiro, etc.) — o código/programação dos apps fica salvo separadamente no GitHub, com histórico de versões automático.
